# My Daily Log / 日々の棚

今日の断片を、形式を決めずに残し、あとから使える記録へ育てるローカル完結型PWA。

## 入っている機能

- 日付ごとのログ、タイトル、気分、持ち越しメモ
- 自由入力とクイック入力
- 食事・支出・運動・ゲームなどへのルールベース分類提案
- ログ詳細の自動保存編集（本文、日付・時刻、種類、数値・単位、タグ、重要度）
- 「特大イベント」の前後3日を一続きで読むイベントビュー
- カレンダー、月別集計、全文検索
- IndexedDBのローカル保存、JSONバックアップ／復元、CSV出力
- PWAオフライン対応、ライト／ダークモード

## ローカルで起動

```bash
npm install
npm run dev
```

## GitHub Pagesで公開

`npm run build` で生成される `docs/` を公開対象にする。
GitHub の **Settings → Pages** で、以下を選択する。

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/docs`

このアプリのログ本体はブラウザのIndexedDBにだけ保存され、リポジトリには保存されない。ブラウザのデータ削除や端末変更に備え、設定画面から定期的にJSONバックアップを取る。
